Информация о разрешениях

  • Server

    1. Чтение с sd-card необходимо для выполнения lua-скриптов.
    2. Запись на sd-card необходима для работы с файлами, лежащими по пути, указанном в настройках.
    3. Интернет необходим для нормального функционирования сервера.

  • Editor

    1. Чтение с sd-card необходимо для выполнения lua-скриптов.
    2. Запись на sd-card необходима для записи lua-скриптов.

  • Client

Функции lua

  1. print(text) - печатает текст клиенту, вызвавшему скрипт

Объекты и функции сервера

Внутри скрипта можно получить ссылку на некоторые объекты и функции сервера. Объекты можно использовать сразу же, как будто они были созданы в скрипте, а функции выцываются так: объект : его_функция. Примеры использования можно посмотреть в готовых скриптах.
  1. server - ссылка на поток lua.
  2. server:send2all(String text) - отправляет text всем игрокам сервера. Ничего не возвращает.
  3. server:getAnswer() - Возвращает строку с ответом клиента.
